# Badminton Corner Randomizer

A lightweight browser-based interval trainer for badminton footwork, shadow training, conditioning, and movement drills.

The app displays large random movement directions with configurable timing, countdowns, beeps, fullscreen support, and training blocks optimized for phone usage in gyms or sports halls.

Designed primarily for badminton players, but also useful for:
- squash
- tennis
- table tennis
- padel
- general agility training
- HIIT / conditioning circuits

## Features

### Random movement directions
The app displays 6 movement directions:
- Front Left
- Front Right
- Left
- Right
- Back Left
- Back Right

Directions are weighted to mimic more realistic badminton movement patterns:
- 50% front ↔ back transitions
- 30% same-area transitions
- 20% lateral transitions

## Training features

### Configurable intervals
You can configure:
- movement duration
- micro-break duration between movements
- directions per set
- number of sets
- active break duration
- passive break duration
- countdown duration
- beep frequency

### Active breaks
Active breaks can be labeled individually, e.g.:
- ROPE
- CORE
- MOBILITY
- SHADOW
- BURPEES

This allows combining footwork with conditioning circuits.

### Countdown system
The app includes:
- start countdown
- set countdowns
- audio countdown beeps
- distinct final countdown sound

### Fullscreen mode
Optimized for phone fullscreen usage with:
- huge arrows
- large readable labels
- landscape support
- stable non-jittering timers
- color-coded directions
